# 01 — Mac / Apple Administration

## Purpose

This folder contains labs for supporting and administering Apple devices in enterprise or mixed Windows/macOS environments.

The labs should build confidence with macOS support, Apple device management concepts, user communication, and escalation-quality evidence.

## Topic Focus

Labs in this folder may cover:

* macOS first-line and second-line support
* local users and permissions
* FileVault
* Keychain
* Gatekeeper
* software updates
* Wi-Fi, VPN, DNS, proxy, and certificate issues
* Apple Business Manager
* MDM enrolment
* configuration profiles
* app deployment
* Microsoft 365 and identity integration

## Mandatory Career Layers

Every Mac / Apple administration lab must also include:

* structured troubleshooting logic
* user-friendly communication
* manager or escalation update
* interview explanation
* personal branding output
* reflection on confidence and professionalism

## Example Lab Folder Names

```text
lab-01-enterprise-mac-access-issue/
lab-02-filevault-and-keychain-support/
lab-03-macos-network-troubleshooting/
lab-04-mdm-profile-review/
```

## Evidence Standard

Mac support evidence should be documented clearly but safely. Do not commit serial numbers, real user names, asset tags, screenshots with private information, company data, or live configuration exports.
